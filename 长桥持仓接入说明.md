# 长桥(Longbridge)真实持仓 · 接入与自动更新说明

网站持仓表已接入**真实长桥持仓**（含期权），并显示**盈亏路程**。数据流：

```
长桥 MCP 服务  ──(只读 token)──▶  同步脚本(纯标准库)  ──▶  data/holdings.json + data/holdings_history.json  ──▶  网站读取展示
```

- 真实持仓快照已发布（首次由本地同步生成并提交）。
- 自动更新由 GitHub Action 完成（见下，需配 1 个 Secret）。

> ⚠️ 你已选择「**完全公开真实持仓**」：你的真实仓位、市值、盈亏（含期权、约 −29% 浮亏、净值结构）对**所有访客可见**，且会被搜索引擎/缓存**永久留存**。要撤回：删除 `data/holdings.json` 和 `data/holdings_history.json` 并停用工作流。

---

## 🔐 安全：token 会泄露吗

**正确设置下不会**泄露到公开仓库/网站/数据文件：
- token 存 GitHub **加密 Secret**，工作流只引用 `${{ secrets.X }}`，明文不进代码。
- 同步脚本 `scripts/sync_holdings.py` **只输出持仓字段**，绝不打印/写入 token。
- GitHub 自动把 Secret 在日志里替换为 `***`。
- 你的 token **scope 是只读**（`watchlist / account.read / trade.read`，无下单）——万一泄露别人也只能看，不能交易。

## 开启自动更新（配 1 个 Secret）

### 第 1 步：拿到 MCP access token
在 https://open.longbridge.com/connect 生成一次性授权码，再用它换 access token。
> 本次接入时我已帮你换好并临时存于本机 `F:\vscode\_tools\lb_token.txt`（**有效期到约 2026-07-04**）。直接打开该文件复制那串 token 即可。

### 第 2 步：把 token 存进 GitHub Secret
仓库 → `Settings` → `Secrets and variables` → `Actions` → `New repository secret`：

| Name（必须一致） | Value |
|---|---|
| `LONGBRIDGE_MCP_TOKEN` | 上面那串 access token |

### 第 3 步：手动跑一次确认
仓库 → `Actions` → `Sync Longbridge Holdings` → `Run workflow`。成功后 `data/holdings.json` 会被更新，日志里 token 显示为 `***`。之后按 cron（交易时段每 30 分钟）自动更新。

---

## ⚠️ Token 续期（重要）

MCP token **约 2 周过期**。过期后 Action 会失败，需要：
1. 在 https://open.longbridge.com/connect 重新生成授权码 → 换新 token；
2. 回到第 2 步更新 `LONGBRIDGE_MCP_TOKEN` 这个 Secret。

**嫌麻烦的替代方案**：
- **让我按需刷新** —— 你随时丢给我一个新授权码，我重新拉取并提交，**token 完全不进 GitHub**（最安全，零维护，但非自动）。
- **改用 OpenAPI App 密钥**（开发者后台建只读 App，Access Token 90 天）—— 更耐久，但需改回 SDK 方案，告诉我即可切换。

## 文件清单

| 文件 | 作用 |
|---|---|
| `scripts/sync_holdings.py` | 同步脚本（MCP HTTP，纯标准库）：拉持仓+期权/股票行情+汇率 → 写数据文件 |
| `.github/workflows/sync-holdings.yml` | GitHub Action：定时/手动，注入 token secret，自动提交 |
| `data/holdings.json` | 真实持仓快照（公开）。期权带 `multiplier/strike/expiry` |
| `data/holdings_history.json` | 盈亏路程历史（每天一个点，自动追加） |
| `holdings.js` / `index.html` | 前端：只读展示真实持仓（期权 ×100）+ 盈亏路程曲线；无数据时回退可编辑示例表 |

**多币种/期权**：每只按本币显示，期权市值 = 张数 × 权利金 × 合约乘数(100)；总额按当日汇率折算人民币(¥)。

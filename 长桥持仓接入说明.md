# 接入长桥(Longbridge)真实持仓 · 设置说明

让网站持仓表**自动显示你的真实长桥持仓**：GitHub Action 定时用长桥 OpenAPI 抓取持仓+行情 → 写成 `data/holdings.json` → 网站读取展示。

> ⚠️ 你已选择「**完全公开真实持仓**」：接入后你的仓位/市值/盈亏会对**所有访客可见**。如果改主意，删掉 `data/holdings.json` 并停用工作流即可回到手动可编辑表。

---

## 🔐 安全：token 会泄露吗？

**正确设置下不会**泄露到公开仓库/网站/`holdings.json`：
- 凭据存 GitHub **加密 Secrets**，工作流只引用 `${{ secrets.X }}`，明文不进代码。
- 同步脚本**只输出持仓字段**（名称/代码/股数/成本/现价/昨收/币种），**绝不打印或写入 token**。
- GitHub 自动把 Secret 在 Actions 日志里替换为 `***`。

**但有一条必须做对（关键防护）：**

> ### 🔑 建 App 时只开「只读」权限——只勾「行情」+「账户资产」，**绝不勾「交易/下单/撤单」**。
> 这样即使 token 万一泄露，别人也**只能看你的持仓**（你已选择公开），**无法交易或提现**。

## 三步设置（一次性）

### 第 1 步：在长桥建一个「只读」App

1. 打开 https://open.longbridge.com → 登录 → 开发者后台 / 用户中心。
2. 新建 App（或用现有的），在权限里**只勾选**：
   - ✅ 行情（Quote）
   - ✅ 账户资产 / 持仓（Account Assets）
   - ❌ **不要勾** 交易 / 下单 / 撤单（Trading）
3. 拿到三个凭据：**App Key**、**App Secret**、**Access Token**。
   > Access Token 有有效期，过期后需在后台重新生成并更新下面的 Secret（见末尾「续期」）。

### 第 2 步：把凭据存进 GitHub Secrets（加密，不会公开）

仓库页面 → `Settings` → `Secrets and variables` → `Actions` → `New repository secret`，**逐个**添加：

| Name（必须完全一致） | Value |
|---|---|
| `LONGPORT_APP_KEY` | 你的 App Key |
| `LONGPORT_APP_SECRET` | 你的 App Secret |
| `LONGPORT_ACCESS_TOKEN` | 你的 Access Token |

### 第 3 步：手动跑一次，确认成功

1. 仓库 → `Actions` 标签 → 左侧选 `Sync Longbridge Holdings` → 右侧 `Run workflow` → 绿色按钮。
2. 等 1 分钟看运行结果：
   - ✅ 成功后仓库会多出 / 更新 `data/holdings.json`。
   - 打开它确认：**只有持仓数据，没有任何凭据**。
   - 看运行日志：凭据应显示为 `***`，无明文。
3. 打开网站，持仓表会自动切换成**只读的真实持仓**（标注「来自长桥」）。

之后它会按 `.github/workflows/sync-holdings.yml` 里的定时（A股/美股交易时段每 30 分钟）**自动更新**，你不用管。

## 它是怎么工作的（文件清单）

| 文件 | 作用 |
|---|---|
| `scripts/sync_holdings.py` | 同步脚本：抓持仓+行情+现金，折算汇率，写 `data/holdings.json`。只输出持仓，绝不碰凭据。 |
| `.github/workflows/sync-holdings.yml` | GitHub Action：定时 + 手动触发，注入 Secrets 跑脚本并提交结果。 |
| `data/holdings.json` | 生成的真实持仓数据（公开）。网站读取它。 |
| `holdings.js` / `index.html` | 前端：有 `holdings.json` 就只读展示真实持仓；没有就回退到内置可编辑示例表。 |

**多币种**：你的持仓跨 A股(¥)/美股($)等。每只按**本币显示**价格/市值；总资产、盈亏按**当日汇率折算成人民币(¥)**汇总（汇率来自免费源 frankfurter.app）。

## 常见问题

- **现价不对/为成本价**：说明 App 没开「行情」权限，或该市场行情需先在长桥开通（多为免费）。持仓接口只给成本价，现价靠行情接口。
- **定时不触发**：GitHub 定时任务可能延迟几分钟；且仓库 **60 天无提交会自动停用定时**——偶尔手动 Run 一次或有提交即可保活。
- **Token 续期**：Access Token 到期后，在长桥后台重新生成，回到第 2 步更新 `LONGPORT_ACCESS_TOKEN` 这个 Secret 即可（旧 token 作废）。
- **想关掉/回到手动**：删除 `data/holdings.json`（并可停用该工作流），网站即回到内置可编辑持仓表。

> 投资有风险，入市需谨慎，本站仅供参考不构成投资建议。

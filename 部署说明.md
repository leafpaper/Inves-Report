# 部署说明 · 叶纸的投资报告

这是一个**纯静态网站**：没有后端、不需要打包构建、所有依赖（React / Babel / 字体）都已下载进仓库，**国内可直接访问，不依赖任何外网 CDN**。

---

## 一、本地预览（不用装任何东西）

直接**双击 `index.html`**，用浏览器打开即可。所有功能（持仓编辑、搜索筛选、报告跳转）都能正常用，因为全是本地文件。

> 持仓数据保存在你浏览器的 localStorage 里，换电脑/换浏览器不会同步——用持仓编辑栏里的「⬇ 导出备份」存一份 JSON，换设备时「⬆ 导入」即可。

---

## 二、上传到 GitHub（推荐用 GitHub Desktop，全程图形界面，不用命令行）

> 你这台电脑没装 git 命令行，所以**用 GitHub Desktop 最省事**（它自带 git，纯点鼠标）。
> 下载：https://desktop.github.com/

### 情况 A：放进你已有的仓库（替换旧版网站）

1. 打开 GitHub Desktop → `File` → `Clone repository` → 选你已有的仓库（如 `Inves-Report`）→ 克隆到本地某文件夹。
2. 把本项目 `qaq` 文件夹里的**所有内容**，复制粘贴进刚克隆下来的那个仓库文件夹（覆盖旧文件）。
3. 回到 GitHub Desktop，左侧会列出所有改动 → 在左下角填一句话（如 `升级二次元设计 + 自托管依赖 + 持仓可编辑`）→ 点 `Commit to main`。
4. 点右上角 `Push origin`。完成上传。

### 情况 B：新建一个独立仓库

1. GitHub Desktop → `File` → `New repository` → 名字随便起（如 `leafpaper-report`）→ Local path 选一个空文件夹 → `Create repository`。
2. 把 `qaq` 文件夹里**所有内容**复制进这个新仓库文件夹。
3. `Commit to main` → `Publish repository`（取消勾选 "Keep this code private" 才能开 Pages）。

---

## 三、开启 GitHub Pages（让别人能在线访问）

1. 浏览器打开你的仓库 → `Settings`（设置）→ 左侧 `Pages`。
2. `Source` 选 `Deploy from a branch`；`Branch` 选 `main`，文件夹选 `/ (root)` → `Save`。
3. 等 1–2 分钟，页面顶部会出现网址，形如：
   `https://你的用户名.github.io/仓库名/`
4. 打开它就是你的线上网站。首页 = `index.html`，会自动加载。

---

## 四、上线后记得改一处

`index.html` 里有一行配置（约在 body 中部）：

```html
<script>window.SITE = { github: 'https://github.com/leafpaper/Inves-Report' };</script>
```

把里面的网址改成**你自己的 GitHub 仓库地址**，导航栏和页脚的「GitHub」按钮就会指向你的仓库。

## 五、目录速览

| 路径 | 作用 |
|---|---|
| `index.html` | 网站首页（报告聚合 + 可编辑持仓）——GitHub Pages 入口 |
| `report.html` | 个股分析页，按 `?slug=` 渲染（盛美上海为完整样例） |
| `holdings.js` | 增强版持仓表（手动编辑 / 自动算盈亏占比 / 导出导入） |
| `reports.data.js` | 报告与持仓的示例数据，**改这里**增删标的 |
| `vendor/` | 自托管的 React / Babel（不依赖 unpkg） |
| `tokens/fonts/` | 自托管字体 woff2（不依赖 Google Fonts） |
| `styles.css` `tokens/` `_ds_bundle.js` `assets/` | 设计系统与组件、素材 |

---

## 六、怎么换成你的真实数据

- **报告列表**：编辑 `reports.data.js` 里的 `window.REPORTS` 数组（每个标的一项）。
- **持仓**：上线后直接在网页上点「✎ 编辑持仓」改即可（存本地浏览器）；要改默认值就改 `reports.data.js` 里的 `window.HOLDINGS`。
- **吉祥物立绘**：把你的角色图替换 `assets/mascot-placeholder.jpg`（正方形 / 透明 PNG 最佳）。

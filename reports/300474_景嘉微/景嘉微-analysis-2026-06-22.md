### 行业规模与趋势

景嘉微处于 **半导体 / 国产 GPU + 军工电子** 交叉赛道，Tushare 行业分类为"元器件"（全行业 306 家上市公司）。行业规模与增长驱动（引用外部数据源）：

- **全球半导体市场**：管理层引用世界半导体贸易统计组织（WSTS）预测，**2025 年半导体市场规模将达 6,970 亿美元，增长率 11.2%** [PDF:2024_annual, P.11]。
- **中国 IC 设计行业**：**2024 年中国集成电路设计行业销售额预计为 6,460.4 亿元，同比增长 11.9%** [PDF:2024_annual, P.11]——景嘉微作为 Fabless 设计企业直接受益于该赛道高景气。
- **边端 AI 芯片市场**：2026 年预计达 **500 亿元**（phase1-data.md §8）。景嘉微差异化路线是避开寒武纪/海光的数据中心训练正面竞争，主打"GPU + 边端 AI SoC"。
- **行业生命周期**：国产 GPU/AI 算力处于**成长早期—规模化落地关键期**。2026 年国产 AI 算力从"替代进口"转向"性能优化"，信创/智算中心/边缘 AI 三场景突破（phase1-data.md §8）。

### 4.3 定性综合判断（逻辑三段式，无打分数字）

> 估值分析见 §五。会计/治理红旗集中在 §六 审计（Buffett Quality + Sloan Accrual + Valuation Anomaly 三框架已自动扫描）。本节聚焦不能被定量指标替代的"深层判断"，verdict 作为 §七 7.1 状态后验的**输入之一**。

#### 框架 1：护城河判断

**核心问题**：景嘉微的可持续竞争优势在加深、侵蚀还是部分是？

**判断**：部分是（一硬一软，强度悬殊）

**证据链**：
1. 军工图形显控市占率 >50%、军工资质 + 大基金背书 = 静态硬壁垒，过去由 62.91% 行业第一档毛利率证明其议价权[PDF:2024_annual, P.78 / Tushare]。
2. 国产 GPU 通用化身处红海：寒武纪 2026Q1 +159.56% 已扭亏、海光 DCU 规模化部署，生态壁垒（CUDA）在对手侧，景嘉微商业化进度落后（反例/侵蚀面）[peer 业务可比组]。
3. 存疑点：景宏 AI 分部独立业绩未披露，无法验证第二曲线护城河真实形成度（§九 盲区）。

**逻辑蕴含**：判断"部分是" → 强制 SOTP 分部估值（成熟军工显控按硬件倍数、GPU 按期权/AI 同业倍数），不可单一笼统倍数盖全公司。

## §五 估值、赔率与定价充分度

> **方法论（v7.0）**：不以"DCF 概率加权单锚"为唯一说服力。TTM 净利为负 → **PE 不适用（audit 🟠 红旗 #3）**，主用 **P=F+N 分解 + 反向 DCF 隐含预期 + 叙事分部 SOTP** 判赔率/定价充分度，正向 DCF 仅作 F 的交叉验证。现价 55.11 元（20260622）、总股本 5.226 亿股、市值 288.02 亿。

### 5.1 价格分解 P = 基本面 F + 叙事溢价 N

**F（基本面锚）= 近端可见业务保守估值，区间 40-49 亿，取中值 44 亿（每股 ≈8.4 元）**：
- 下界 = 净清算地板 **39.92 亿 / 7.64 元**（货币 31.02 + 应收 7 折 4.46 + 存货 5 折 2.73 + 固定/无形 5 折 5.25 + 景美子公司净资产 8.43 − 总负债 9.38 − 清算成本 2.59）[§二 SOTP]。
- 上界 = 持续经营口径 **≈48.6 亿**：军工成熟业务（显控+雷达，约 4.5 亿成熟营收 × 6x PS = 27 亿，军工 Fabless 高毛利但成熟）+ 净现金 21.64 亿。
- 取中值 **F ≈ 44 亿**。

**N（叙事溢价）= 现价市值 − F = 288.02 − 44 = 244 亿，占现价市值 ≈84.7%**。

- **N 在为什么定价**：国产 GPU/AI 算力自主可控右尾期权（JM9/景宏放量 + 太空算力/星网订单 + 政策红利）。市场把 PS_TTM 推到 41.04（行业 96.4 分位）即为此期权定价。
- **N 是 free option 还是 embedded obligation？→ embedded obligation（必须交付的义务）**。在 PS 96.4 分位的高估值下，GPU 放量不是"免费看涨期权"，而是市场已要求必须兑现的义务——若 2026 中报 GPU 收入确认落空，N 将大幅回吐（向 F 收敛 = 现价 −80%+）。
- **N'/N vs F'/F**：驱动股价的是叙事增速（AI 主题资金/政策催化，N' 大）而非基本面增速（F' 受持续亏损压制接近零甚至为负）→ **N'/N > F'/F**，落入"信念越强、股票越贵、回报越差"的数学死穴。

### 5.2 反向 DCF：市场隐含预期

从现价 288 亿市值倒推市场隐含假设：

| 路径 | 隐含假设 | vs 历史/同业参照 | 可信? | 证伪条件 |
|---|---|---|:---:|---|
| PS 维持 12x | 营收需 24.0 亿（现 7.2 亿，3 年 CAGR ≈49%） | 历史峰值仅 11.54 亿（2022），从未达 24 亿 | 🔴 过度乐观 | 中报 GPU 收入确认 < 预期 / 营收 3 年未破 15 亿 |
| PS 维持 15x | 营收需 19.2 亿（3 年 CAGR ≈39%） | 需超 2022 峰值 66% | 🟠 激进 | 同上 |
| PS 维持 18x | 营收需 16.0 亿（3 年 CAGR ≈30%） | 需超 2022 峰值 39% | 🟡 需 GPU 放量支撑 | GPU 营收占比 3 年未过 30% |
| 盈利端 | 净利率 20% + 退出 PE 30 → 需净利 9.6 亿 / 营收 48 亿（6.7x 现） | 历史峰值净利仅 2.89 亿（2022） | 🔴 极度乐观 | 终端净利率 < 15% 或营收 < 30 亿 |

**结论：现价隐含预期 = 买完完美未来（接近）**。即便给国产 GPU 同业级别的高 PS（15-18x），也需营收 3 年 39-49% 复合增长且突破历史峰值——对一个仍在续亏、GPU 尚未规模放量的公司，这是市场已"提前透支"的乐观假设。证伪点集中在 2026 中报 GPU 分部收入确认。

**路径可承受性初判**（供 §七 7.3）：**高尾险**。理由：①兑现前需持续承受续亏 + 年烧研发 1.6 亿+（时间成本不免费）；②左尾"叙事证伪 −80%+"概率中高（30%）；③高信仰特征 3 强命中 → 下跌放大；④唯一缓冲是清算地板（7.64/4.14 元）提供"亏损保护但非上行保护"。**路径不可在现价满仓承受，仅可在小仓位/证据临界后承受**。

## §七 投资决策内核

> 用"**投资价值 = 状态后验 × 赔率 × 路径可承受性**"（乘法，任一为"差"即拉低整体）合成 §四 4.11（状态）、§五（赔率）、§六 6.4（路径），输出"好公司 / 好下注 / 好价格"三分结论与行动档位。本章是各篇理念的合成结论与全报告 punch line。框架定义见 `references/investment-decision-core.md`。

### 7.1 状态后验合成（引 §四 4.11）

景嘉微是"财报型军工基本盘 + AI receiver 第二曲线"混合体，λ（向上跳变强度）需分部看：

- **军工基本盘 λ↑ 且证据独立**：FY2025 营收 +54.41% V 反转、2025Q3 单季营收 +230.65% 且净利转正 +1,508 万、毛利率维持 62.91% 行业第一档——**这些是实际财报数据（独立证据✓），但已被部分定价、且被全年续亏 −1.65 亿稀释**。
- **GPU/AI 第二曲线 λ 微弱且为回声**：JM11 仅流片/小批量交付（未见规模收入）、研发 +102.67% 是"成本跳变非收入跳变"、景宏 AI 分部独立营收未披露（盲区）；"国产 GPU 龙头/太空算力放量"是政策口号 + 券商转述（回声✗）。**承载 84.7% 高估值的恰是这条尚未被独立证据确认的曲线。**
- **贝叶斯三问**：① 后验提高还是注意力提高？→ **主要是注意力**（AI 主题 ETF 新进、政策催化），GPU 业务后验未被独立证据提高；② 新证据独立还是回声？→ 军工独立、GPU 回声为主；③ 价格是否过度反映后验？→ **是**（PS 96.4 分位）。
- **无记忆性检查（Rule 7）**：现价距 60 日支撑仅 8%、近 1 年 −22.42%、PB 近 1 年 5% 分位——**禁止把"跌久了该反弹 / PB 压到历史低位该修复 / 国产替代讲多年该兑现"当买入理由**。唯一合法买入触发 = λ↑ 或证据斜率转正，当前 GPU 斜率未转正（投入非产出）。

→ **状态后验**：**↑变好但仅注意力·未确认**（军工基本盘确有向上跳变且证据独立，但核心高估值叙事的"变好"仅被注意力/政策提高，未被独立业务证据确认；临界点 = 2026 中报 GPU/AI 分部收入披露）。

### 7.2 赔率合成（引 §五）

- **P=F+N（5.1）**：F≈44 亿（每股 ≈8.4 元）、N=244 亿（占现价市值 ≈84.7%），且 **N 是 embedded obligation（必须交付的义务）而非 free option**；N'/N > F'/F（叙事增速驱动 > 基本面增速）→ 落入"信念越强、股票越贵、回报越差"的数学死穴。
- **反向 DCF（5.2）**：现价隐含营收 3 年 CAGR 39-49%（需突破 2022 历史峰值 11.54 亿达 19-24 亿）/ 盈利端需净利 9.6 亿（历史峰值仅 2.89 亿，6.7x 现）——**买完完美未来（接近）**。
- **SOTP（5.4）+ 正向 DCF（5.5）交叉验证**：分部加总 ≈59-89 亿（每股 11-17 元）、概率加权每股 ≈34.05 元，**均 < 现价 55.11 元**；概率加权 12-18 月期望 **−38%**（乐观 +25%/基准 −46%/悲观 −78%/最差 −86%，概率 25/45/25/5）。
- **ΔP 分解（5.3）**：以"叙事压缩 + 注意力 + 同业共振"驱动为主，叠加"估值耗尽 + 主力资金净流出"强负向——非"业务真相级权威认证"驱动。

→ **赔率**：**买完完美未来**（无上行 slack——上行已被价格吸收；仅下行有清算地板 7.64 元/纯现金 4.14 元托底，提供"亏损保护非上行保护"）。

### 7.3 路径可承受性合成（引 §六 6.4 + §五 5.6）

- **左尾（6.4）**：GPU 通用化证伪 −80%+（概率中高 30%）、减值 / 补助退坡 / 散户踩踏并存；控制权丧失 / 退市基本排除（Z=19.21、净现金、零非标）。
- **高信仰特征（6.4）**：5 项命中 3 强（高估值 / 高波动 / 高散户）+ 2 部分——下跌时"组合波动放大器"，无机构托底。
- **时间 / 机会 / 假阳性成本（5.6）**：兑现临界点在 2026 中报后，兑现前持续承受续亏 + 年烧研发 1.6 亿+（时间不免费）；100 万锁定在赔率为负 + 技术面偏空标的（机会成本）；把"国产替代回声"当 GPU 业务真相买入若证伪，代价 = 现价 −80%+（假阳性成本）。
- **技术面（5.6）**：空头排列（MA5/20/60/120 = 54.31/56.79/60.29/66.24）、跌破 MA60/MA120 → "下行风险放大"象限。

→ **路径**：**高尾险**（左尾毁灭概率中高 + 高信仰特征放大下跌 + 时间成本持续；唯有清算地板提供有限亏损保护。现价满仓不可承受，仅小仓位 / 证据临界后可承受）。

### 7.4 决策合成

> 价值 = 状态后验 × 赔率 × 路径。三者乘积：↑未确认 × 买完完美未来 × 高尾险 = **三个维度都偏"差"，乘积显著为负**。

- **决策三元组**：**[状态后验: ↑变好但仅注意力·未确认 | 赔率: 买完完美未来（无上行 slack） | 路径: 高尾险]**

- **三分结论**（好公司 ≠ 好下注 ≠ 好价格，三个分开判）：
  - **好公司? — 部分是（一硬一软）**。依据：军工图形显控市占率 >50%、毛利率 62.91% 行业第一档、大基金 5.00% 背书、Z=19.21 财务极稳健 = 真护城河基本盘；但 GPU 第二曲线身处"红海中的红海"（寒武纪已扭亏 +159.56%、海光规模化部署、CUDA 生态在对手侧），且连续两年亏损、ROE −2.4%、自由现金流为负——是"好基本盘 + 未成型第二曲线"的混合体，非纯粹好公司。
  - **好下注（现价）? — 不是**。依据：状态×赔率×路径乘积显著为负——概率加权 12-18 月期望 −38%，状态未确认、赔率买完完美未来、路径高尾险，三维度同向偏负，现价 55.11 元下注是"用回声当证据买入透支的未来"。
  - **好价格? — 不是（无 pricing slack）**。依据：N 占现价市值 84.7% 且是 embedded obligation；反向 DCF 显示需营收 3 年 39-49% 复合增长且破历史峰值；SOTP（11-17 元）/ 概率加权（34 元）均远 < 现价 55.11 元。现价非"贵但有 slack 的未定价迁移"，而是"贵且买完完美未来"。

- **行动档位**：**等证据临界（观察）**（六档之一）— **映射依据**：状态后验 ↑未确认 + 赔率买完完美未来 + 路径高尾险 → 按 `investment-decision-core.md` 映射表落"等证据临界（观察）或期权仓（小）"区间；又因右尾真实存在（国产 GPU/AI 需求真实）但景嘉微非"幂律生成器节点"（瓶颈在制程/CUDA/海光寒武纪），右尾是 beta 非 alpha，故**现价不给期权仓、先观察**。**100 万的具体纪律（与 §五 5.6 一致）**：现价 55.11 元 **0 仓**（不追高、不满仓）；100 万作"等证据临界的观察资金 / 期权仓预算"——**若 2026 中报 GPU/AI 分部收入确认放量（证据斜率转正），启用期权仓 ≤15 万（≤15%）博右尾**（叙事越远仓位越小）；若回落至净清算地板附近（7.64 元区，安全边际充分）则风险收益反转，可重估。**这是"高质量等待"而非"资本钝化"——因为有明确等待事件 + 明确退出条件，不是"等久了该涨"的套牢自我安慰。**

- **该等什么**（跳变事件 + 预计时点）：**2026 中报（约 2026-08）GPU/AI（JM11/景宏）分部营收与收入确认明细**——这是把"通用化"从叙事回声变独立证据的**唯一临界事件**。次级观察点：FY2025 年报全文核"38.33 亿定增募集资金使用情况"（是否被理财消耗）；FY2026 业绩预告（约 2027.1，若研发拐头则首现减亏信号）。

- **证伪 / 退出条件**（每条核心叙事 1 个可证伪指标 → 接 Phase 7 监控）：
  - 叙事①"GPU 通用化放量"→ **证伪：2026 中报 GPU/AI 分部营收 < 2 亿 或 研发持续 +100% 而无营收对应** → N 向 F 回吐、叙事崩，退出 / 不启用期权仓。
  - 叙事②"军工基本盘复苏延续"→ **证伪：军工收入确认连续两中报不及预期 或 毛利率跌破 50%** → 基本盘 F 下移。
  - 叙事③"定增资金投向 GPU 研发"→ **证伪：年报披露募集资金大额配置理财 / 收购消耗** → 管理层"才"证伪、估值再折价。
  - **退出触发**：上述任一证伪指标命中 / λ 下降 / 证据斜率转负 / 左尾事件（大额减值、补助退坡）发生 → 不入场或减仓。

- **信仰陷阱五弊端自检**：
  - ① 好公司=好股票？→ **已隔离**：好公司（部分是）≠ 好下注（不是）≠ 好价格（不是），三分独立判，未把"军工护城河好"等同于"现价该买"。
  - ② 远期可选项当当期现金流？→ **已隔离**：明确 N 是 embedded obligation 非 free option，GPU 期权价值仅在 SOTP 中给 10-40 亿且要求证伪指标，未把远期 GPU 收入当已实现。
  - ③ 坏消息当短期噪音、失去证伪？→ **已保留证伪**：续亏 / OCF 红旗 / 主力撤退 / 大宗折价均作为结构性信号纳入空头链，未当"短期噪音"忽略；设了 3 条可证伪指标。
  - ④ 时间当免费、忽略机会成本？→ **已计入**：明确时间成本（兑现前年烧 1.6 亿）+ 机会成本（100 万锁定负赔率标的）+ 假阳性成本。
  - ⑤ 仓位由信念强度而非赔率证据决定？→ **已隔离**：现价 0 仓，仓位由"证据斜率是否转正"决定（中报放量才启用 ≤15 万期权仓），而非"国产替代信仰"强度。

### 7.5 与 §一 对齐

> §一「决策结论」的决策三元组 / 三分结论 / 行动档位必须与本节 7.4 一致，可直接照抄：
> - **决策三元组**：[状态: ↑变好但仅注意力·未确认 | 赔率: 买完完美未来 | 路径: 高尾险]

> - **该等什么**：2026 中报（约 2026-08）GPU/AI 分部营收与收入确认明细
